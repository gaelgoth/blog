# gaelgoth/blog

Source code of blog: <https://blog.gothuey.dev/>

## Developing

Requires Node.js 24.x (upcoming LTS) and pnpm 10.

Start a development server:

```bash
pnpm dev
```

## Building

Create a production version of ur blog:

```bash
pnpm build
```

Preview build app:

```bash
pnpm preview
```
